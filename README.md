

# 🟢 Hacking Terminal v2.0

GitHub で公開中の
**“完全に合法なハッカーごっこターミナル”**

リアルなハッキングっぽさを演出するだけのフロントエンドアプリです。
実際には何もハックしません。安心して遊べます（むしろそれが限界）。

---

## ✨ Demo

ブラウザで開くだけで即起動するターミナルUI

* Matrix風バックグラウンド
* 偽コマンドインターフェース
* ハッキング演出シーケンス
* リアルタイム風ログ表示

---

## 📟 Available Commands

```
help      - コマンド一覧表示
clear     - 画面クリア
date      - 現在時刻
whoami    - ユーザー情報
echo      - 文字出力
hack      - ハッキング風演出（重要）
status    - システム状態表示
```

---

## 💀 hackコマンドについて

このプロジェクトの本体。

実行すると以下の“それっぽい流れ”が発生します：

* IPスキャン（してない）
* パスワード総当たり（してない）
* データダウンロード（してない）
* ログ消去（してない）

全部フィクションです。
あなたのPCは1ミリも侵害されません。
